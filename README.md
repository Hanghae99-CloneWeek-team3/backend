## 📚프로젝트 소개

이미지, 아이디어를 올리고 공유하고 저장하는 사이트인 핀터레스트 클론 코딩을 진행해보았습니다.
게시글을 올리고 댓글도 달고 마음에 드는 댓글에 좋아요를 눌러 소통이 가능하게 만들었습니다.


## 📆프로젝트 제작 기간

* 2022년 10월 28일 ~ 2022년 11월 3일
